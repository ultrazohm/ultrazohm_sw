======
Altium
======

The following instructions are intended to make it easier to get started with Altium, therefore helpful tips, tricks and useful tutorials are referred to. 
It is also shown how to create a new project in the style of the already existing ones or how to deal with the database for part management in the ELSYS institut.
Within the individual chapters, the respective procedure is clarified by screenshots and explanations.

This documentation is not and is in no way intended to be a complete documentation for Altium. 
Therefore the `Altium documentation <https://www.altium.com/documentation/altium-designer>`_ is proposed.
