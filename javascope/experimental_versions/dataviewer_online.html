UltraZohm Data Viewer

Static, browser-only log inspection for GitHub Pages or local use. Upload up to two CSV files, compare channels, drag time cursors, export selected windows, and keep large logs responsive with WebGL traces plus viewport-aware downsampling.

Layout

Select how many graph panels stay visible. Each panel keeps its own controls and view state.

Files

CSV parsing happens in the browser using a worker. The page reads delimiter-separated logs with a required time column.

Sync

Lock time navigation across all visible graphs. Zooming one graph updates the others to the same time window.
