Installation
============

.. note:: JDK 21 or higher is required.

#. `Download <https://jdk.java.net/>`_ and install the latest stable JDK release for your OS from OpenJDK.
#. Unzip the folder and move it to your Program Files directory (e.g. ``C:\Program Files\Java``)
#. Add your JDK installation folder to your system PATH variable (e.g. ``C:\Program Files\Java\jdk-26\bin`` or newer)

   .. note:: For more information regarding the system PATH, see e.g. https://java.tutorials24x7.com/blog/how-to-install-openjdk-17-on-windows

#. Open your terminal and check the installed Java version with ``java -version``
